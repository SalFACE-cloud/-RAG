# 二次函数

## 顶点式

二次函数的一般形式：$y = ax^2 + bx + c$（$a \neq 0$）

顶点式：$y = a(x-h)^2 + k$，其中顶点坐标为 $(h, k)$。

由一般式配方得顶点式：
$$y = a(x + \frac{b}{2a})^2 + \frac{4ac-b^2}{4a}$$

## 判别式与图像

判别式 $\Delta = b^2 - 4ac$ 决定抛物线与 $x$ 轴交点个数：

1. $\Delta > 0$：两个交点，图像与 $x$ 轴相交。
2. $\Delta = 0$：一个交点（相切），图像与 $x$ 轴相切。
3. $\Delta < 0$：无交点，图像在 $x$ 轴上方或下方。

## 图像性质

- $a > 0$：开口向上，有最小值。
- $a < 0$：开口向下，有最大值。
- 对称轴：$x = -\frac{b}{2a}$

## 例题

求 $y = x^2 - 4x + 3$ 的顶点坐标和对称轴。

解：配方得 $y = (x-2)^2 - 1$，顶点 $(2, -1)$，对称轴 $x = 2$。
