# 等差数列

## 通项公式

等差数列的通项公式为：

$$a_n = a_1 + (n-1)d$$

其中 $a_1$ 为首项，$d$ 为公差，$n$ 为项数。

## 基本性质

1. 相邻两项差为常数 $d$。
2. 若知道首项和公差，可求任意项。
3. 等差数列前 $n$ 项和：$S_n = \frac{n(a_1 + a_n)}{2}$。

## 例题

已知等差数列首项 $a_1=2$，公差 $d=3$，求 $a_{10}$。

解：$a_{10} = 2 + (10-1) \times 3 = 29$。
